Chapter Three: Styling

HTML styling allows programmers to style the text to make it look more attracting. It is similar to CSS (Casscade Style Sheet) however its more into hardcoding into HTML. HTML styling is more into hard-coding into html, which is more time consuming to do than CSS however it is still useful styling documents.

Bold Style

The bold style element will change any text inside the element into a bold text. This is useful if you are planning to make large bold titles or making some text bold in your documents.

Strong Style

The Strong element can also be used to make text bold however it is mostly used for highlighting keywords in documents.
